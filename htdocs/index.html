Spritemapper

CSS Spritemap Generator

Spritemapper is an application that merges multiple images into one and generates CSS positioning for the corresponding slices.

Batteries included. The package consists of a simple command-line tool that "does the job" and a Python library including its own PNG and CSS parser. The choice of writing/bundling this was to stay off 3rd-party requirements. Anybody who has ever had the pleasant job of installing PIL on various platforms should have a pretty good idea about what we're trying to avoid.

There are multiple alternatives to Spritemapper, but they all require a bit too much manual labour, whereas Spritemapper reads your current CSS and replaces background images and position with the generated result.

This technique drastically improves your website's loading speed, effectively reducing bandwidth by downloading fewer images.

Installation

Open Terminal.app, urxvt, xterm or whatever and type easy_install spritemapper. The tarball is available on Spritemapper's PyPI page.

OS X

Open Terminal and type sudo easy_install spritemapper
Enjoy!

Windows

Make sure you have Python installed and that it is in your PATH, also include the 'Scripts' directory inside your python installation if you want to run spritemapper directly from the command prompt.

Pre-built .msi installer

Go to Spritemapper's GitHub download page and download the latest .msi installer and run it.

From Source

You will need Git and Python setuptools

Open Command Prompt and locate a suitable folder where you want the sources and type:
git clone git://github.com/yostudios/Spritemapper.git
cd into your freshly cloned repository cd Spritemapper
and install python setup.py install

Usage

Run spritemapper /path/to/static/css/**.css and it does its magic. Give the old --help a quick look, too.

Command-line

-h, --help
show a help message and exit
-c INI, --conf=INI
read base configuration from INI (see Configuration options)
--padding=N
keep N pixels of padding between sprites
